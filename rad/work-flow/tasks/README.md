## How to Use

1. Each subagent works from their own task file
2. Mark tasks as complete using `[x]` when done
3. Update the Progress Summary above weekly
4. Use the status indicators:
   - `[ ]` Not started
   - `[~]` In progress
   - `[x]` Completed
   - `[!]` Blocked

## Handoff Protocol

When UI/UX completes a phase:
1. Mark all tasks in phase as `[x]`
2. Notify Frontend Developer
3. Frontend Developer can start that phase
